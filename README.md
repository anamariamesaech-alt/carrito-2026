🛒 Carrito de Compras 2026 - React + Bootstrap
📌 Descripción

Este proyecto es una aplicación web desarrollada con React y Bootstrap, que simula un sistema básico de carrito de compras.

Su objetivo principal es poner en práctica conceptos fundamentales del desarrollo frontend, tales como:

Componentes en React
Uso de props
Enrutamiento con React Router
Maquetación con Bootstrap
Organización profesional de carpetas

La aplicación permite visualizar productos, navegar entre diferentes secciones y demostrar el uso de props mediante un componente de avatares.

✅ Visualización dinámica de productos

🏗 Arquitectura del Proyecto

El proyecto sigue una estructura modular organizada por responsabilidades:

src/
│
├── assets/        → Imágenes y recursos estáticos
├── components/    → Componentes reutilizables (ProductCard, Avatar)
├── pages/         → Páginas principales (Inicio, Articulos, Props)
├── data/          → Datos simulados de productos
│
├── App.jsx        → Configuración de rutas
├── main.jsx       → Punto de entrada de la aplicación
└── index.css      → Estilos globales
✔ Organización explicada
assets/: almacena imágenes y archivos estáticos
components/: contiene piezas reutilizables de la UI
pages/: define cada vista o pantalla principal
data/: simula una base de datos de productos
App.jsx: maneja la navegación (React Router)
main.jsx: renderiza la aplicación
index.css: estilos generales
